
## API Reference

#### Get all items

```http
  GET /api/downloadYoutubeVideoAudio?url={videoURL}
```

| Parameter | Type     | Description                |
| :-------- | :------- | :------------------------- |
| `url` | `string` | **Required**. Youtube Video URL |

#### Get item

```http
  GET /api/getVideoInfo?url={videoURL}
```

| Parameter | Type     | Description                       |
| :-------- | :------- | :-------------------------------- |
| `url`      | `string` | **Required**. Youtube Video URL |
